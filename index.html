Selamat Datang di Website Portofolio Abimanyu

Assalamualaikum, perkenalkan nama saya abimanyu zacky nuryanto putra, ini adalah website yang saya buat sendiri untuk Ujian Praktek TIK, B indo dan Broadcasting.

Tentang Website Ini

Ujian praktek adalah program tahunan Al-haraki untuk kelas 9 sebagai syarat kelulusan, Mapel yang saya buat ini adalah gabungan Mata Pelajaran TIK, B indo dan Broadcasting, untuk TIK kita membuat sebuah website dari Kode HTML, untuk B indo, kita membuat TEKS ARGUMENTASI, dan Broadcasting kita membuat Video mengenai IC dari Teks Argumentasi Bahasa Indonesia.

Tujuan Pembuatan Website

Tujuan dari pembuatan website ini adalah Bagian dari Ujian Praktek Komputer yang bertujuan untuk membuat website menggunakan code HTML yang dimana bahasa komputer HTML ini sangat umum di gunakan dah cukup mudah untuk dimengerti bagi pemula untuk belajar website.

Topik Argumentasi dan Alasan Pemilihan

Untuk Topik Argumentasinya saya adalah LIMATIGA STUDIOFOTO, Alasan saya memilih topik ini karena saya MAgang di LIMATIGA FOTOSTUDIO, saya juga lumayan menyukai fotografi, karena itu saya memilih LimaTiga berawal dari passion saya satu satunya yang disukai di sekolah ini ada fotografi, saya memilih fotografi di bagian pertama dan broadcasting di bagian kedua, Alhamdulilah saya dapat Fotografi.

Harapan dari Video
